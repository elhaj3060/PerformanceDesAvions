Pour exécuter le programme, veuillez suivre les commandes suivantes:

Avant d'exécuter les commandes suivantes, il faut installer python sur votre machine locale.
Le site officiel de python, Python.org, contient l'exécutable à installer.

1 - Ouvrir l'invite des commandes  (cmd)
2 - Se deplacer vers le repértoire contenant le dossier du code ( TP4C/ )
3 - Pour exécuter le code pour la question 4, il faut écrire la commande : python TP4C-Q4.py
4 - Pour exécuter le code pour la question du TP4A et B, écrire la commande : python TP4AB.py (qui se trouve dans le dossier TP4AB/)


Si le code pour la question 4 ne fonctionne pas, il faut installer la libraire numpy. Il suffit ,une seule fois, d'écrire dans l'invite de commande la commande suivantes : 
1) pip install numpy
